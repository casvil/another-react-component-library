# Another React Component Library

A customizable React 19 component library built with Vite, Tailwind CSS v4 and TypeScript.

## üöÄ Getting Started

Install the component library in your React project:

```bash
npm install another-react-component-library
```

### Basic Usage

```tsx
import {
  Form,
  FormInput,
  Select,
  Switch,
  Clock,
} from 'another-react-component-library';

const fruitOptions = [
  { value: 'apple', label: 'üçé Apple' },
  { value: 'banana', label: 'üçå Banana' },
  { value: 'orange', label: 'üçä Orange' },
  { value: 'grape', label: 'üçá Grape' },
];

function App() {
  return (
    <div>
      <div style={{ display: 'flex', justifyContent: 'center' }}>
        <Clock format="HH:mm:ss" bordered />
      </div>
      <div>
        <Form onSubmit={(data) => alert(JSON.stringify(data, null, 2))}>
          <FormInput
            name="username"
            label="Username"
            placeholder="Enter your username"
            required
          />
          <Select
            label="Favorite Fruit"
            options={fruitOptions}
            defaultValue="apple"
          />
          <Switch
            label="Enable notifications"
            defaultChecked={true}
            wrapperClassName="space-x-6"
          />
        </Form>
      </div>
    </div>
  );
}
```

## üß© Component Architecture

This library follows **Atomic Design** principles, organizing components into atoms and molecules:

### ‚öõÔ∏è Atoms

Basic building blocks - fundamental UI elements.

- **Avatar** - User profile images
- **Badge** - Status indicators and labels
- **Button** - Interactive buttons
- **Checkbox** - Selection controls
- **Divider** - Visual separators
- **ErrorMessage** - Error displays
- **HelperText** - Contextual information
- **Icon** - Lucide icon wrapper
- **IconButton** - Icon-only buttons
- **Image** - Image display
- **Input** - Text inputs
- **Label** - Form labels
- **Link** - Navigation links
- **ProgressBar** - Progress indicators
- **Radio** - Single selection controls
- **Spinner** - Loading indicators
- **StatusIndicator** - Status displays
- **Switch** - Toggle controls
- **TextArea** - Multi-line text inputs
- **TimeDisplay** - Time formatting
- **Tooltip** - Information overlays
- **Username** - Username displays

### üß¨ Molecules

Component combinations - groups of atoms working together.

- **BadgeAvatar** - Avatar with badge overlay
- **CheckboxGroup** - Multiple checkbox management
- **Clock** - Real-time clock display
- **Form** - Form management with validation
- **FormField** - Input with label and validation
- **FormInput** - Form-connected input
- **InputField** - Enhanced input with states
- **RadioGroup** - Multiple radio management
- **Search** - Search input
- **Select** - Dropdown selection
- **UserPreview** - User info display
